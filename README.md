# MangoShare

MangoShare is a peer-to-peer file sharing web application that allows users to send files directly between browsers using a simple room code — without storing any data on a server.

 ## Features


🔗 Room-based connection (6-digit code)


⚡ Real-time file transfer


📦 Chunk-based file sharing for efficiency


🔒 No server storage (secure & private)


📱 QR code support for quick joining


🎨 Clean and modern UI



## How It Works


User creates a room → gets a unique code


Another user joins using the same code


A direct connection is established


File is split into chunks and sent in real-time


Receiver reconstructs and downloads the file
